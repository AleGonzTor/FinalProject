# FinalProject
"Programación de videojuegos" final project
"Puchipu's Adventure" is a kind of platformer video game. We made it to improve our logic and to better understand conditionals, void functions, matrices, lists, classes, and inheritance.
We didn’t create the assets ourselves; we took them from other sources. We want to thank all the people who made these sounds, sprites, and fonts available for free.

All the sprites used in this game, except for Char, are from: www.kenney.nl
The font from the menu: Copyright 2021 The Pixelify Sans Project Authors (https://github.com/eifetx/Pixelify-Sans)
